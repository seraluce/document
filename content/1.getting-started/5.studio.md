**Nuxt Studio** 模块是一个基于浏览器的界面，用于直接在生产环境中编辑你的 Nuxt Content 网站。通过 GitHub、GitLab 或 Google 认证访问部署站点，并开始在无需本地开发工具的情况下管理内容。

::tip{to="https://nuxt.studio/introduction"}
查看 Nuxt Studio 文档，了解如何安装该模块。
::

:video{controls loop src="https://res.cloudinary.com/nuxt/video/upload/v1767647099/studio/studio-demo_eiofld.mp4"}

**Studio 编辑器** 允许你在生产环境的网站中完全通过浏览器管理内容。无需本地开发工具、Git 命令或终端访问。这非常适合希望在熟悉环境中编辑和预览内容的团队。

## 在生产环境中可视化编辑 Nuxt Content 网站

Nuxt Studio 提供面向 Nuxt Content 网站的**生产环境可视化编辑**。

Studio 最初作为独立的付费平台提供，如今已成为一个**免费、开源且可自托管的 Nuxt 模块**。它让开发人员和非技术编辑人员都能安全地创建和更新内容，而无需离开实时网站。

## 当前功能

### ✨ **TipTap 可视化编辑器**

支持完整 MDC 组件的富文本 Markdown 编辑器。

### 💻 **Monaco 代码编辑器**

用于直接编辑 Markdown（MDC）、YAML 和 JSON 文件的高级代码编辑器。

### 📝 **表单式编辑器**

使用基于集合模式自动生成的表单编辑 YAML、JSON 和 frontmatter。

### 🎨 **Vue 组件 Props 编辑器**

可视化界面可直接编辑 Vue 组件的 props。

### 🔄 **实时预览**

即时预览生产网站中的内容更改。

### 🔐 **多提供商认证**

支持 GitHub、GitLab 和 Google 的安全 OAuth 认证。

### 🔑 **自定义认证**

提供实现自定义认证流程的工具（密码、SSO、LDAP 等）。

### 📝 **文件管理**

在 [content/]{.s2} 目录中创建、编辑、重命名和删除内容文件。

### 🖼 **媒体管理**

集中式媒体库支持 JPEG、PNG、GIF、WebP、AVIF、SVG 等格式。

### 🌳 **Git 集成**

在生产环境中直接提交内容更改，并依赖你的 CI/CD 管道进行部署。

### 🚀 **开发模式**

使用 Studio 界面直接从本地文件系统编辑内容和媒体文件。

### 🌍 **国际化**

支持 17 种语言：AR、BG、DE、EN、ES、FA、FI、FR、ID、IT、JA、NL、PL、PT-BR、UA、ZH、ZH-TW。

## 即将推出的功能

### 📂 **集合视图**

从统一界面管理和导航所有内容集合。

### 🖼 **媒体优化**

直接在编辑器内优化图片和媒体资源。

### 🤖 **AI 内容助手**

获取智能 AI 建议，提升内容质量并加快创作速度。

### 💡 **社区驱动功能**
