欢迎使用 **Docus**，一个与 [Nuxt UI](https://ui.nuxt.com) 深度集成的文档解决方案。

## 什么是 Docus？

Docus 是基于 [UI 文档模板](https://docs-template.nuxt.dev/) 的主题。视觉样式开箱即用，你的重点应放在使用 [Nuxt Content](https://content.nuxt.com) 提供的 Markdown 和 [MDC 语法](https://content.nuxt.com/docs/files/markdown#mdc-syntax) 来编写内容。

## 主要特性

本主题包含一系列旨在提升文档管理的功能：

- **由** **[Nuxt 4](https://nuxt.com)** **提供支持**：采用最新 Nuxt 框架以获得最佳性能。
- **由** **[Nuxt UI](https://ui.nuxt.com)** **构建**：集成完善的 UI 组件套件。
- 通过 **[Nuxt Content](https://content.nuxt.com)** 提供的 **[MDC 语法](https://content.nuxt.com/usage/markdown)**：支持带组件集成的 Markdown 以创建动态内容。
- **兼容** **[Nuxt Studio](https://content.nuxt.com/docs/studio)**：可视化编写和编辑内容，无需 Markdown 知识。
- **自动生成侧边导航**：根据内容结构自动生成导航。
- **全文搜索**：内置搜索功能，便于内容检索。
- **优化排版**：提供精良的排版体验，提高可读性。
- **暗黑模式**：支持用户偏好的暗黑主题。
- **功能丰富**：探索主题以充分发挥其能力。
